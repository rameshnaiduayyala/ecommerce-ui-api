Enterprise E-Commerce API Portal

Available public service routes and active developer tools:

  • 📖 Swagger OpenAPI Specification Interactive REST API playground for developers.
  • 🧪 System Health Check Verifies database pools, memory logs, and server uptime.
  • 🏷️ Product Catalog Fetches catalog items (Supports search, filtering, and page parameters).
  • 🌿 Categories Hierarchy Fetches nested tree structure of product categories.
  • 🔥 Active Brands Lists registered product brands.
  • 📄 CMS Static Pages Fetches system terms, FAQ, and details.
  • ✍️ Blog Feed Lists published blog posts.
  • ❓ Frequently Asked Questions Retrieves sorting list FAQs.
